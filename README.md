# Memórias da Caatinga
**Jogo de terror/suspense desenvolvido em Unity para a disciplina de computação gráfica. Trabalho realizado por Alexandre Selani e Joel Filho.**

Sinopse: Um degradador do bioma da caatinga encara as consequências de seus atos enquanto busca fugir do purgatório, onde se encontra entre a vida e a morte.

No desenvolvimento do jogo, foi necessário aplicar amplamente conhecimento em programação orientada a objetos e boas práticas de programação. Tais conhecimentos foram muito úteis para suportar nossos ideais criativos propostos para o trabalho. 
## Todos os scripts foram desenvolvidos em C#.

# Como executar:
* Para executar o jogo, basta fazer o download na aba de Releases e executar em um ambiente Windows.
